## Convertor for Indian Languages
### Supported formats UTF,WX, Phonemic
### Supported languages

- Telugu
- Hindi
- Tamil
- Kannada
- Oriya
- Punjabi
- Malayalam

### Features supported
- Browser file
- Copy/Paste input
- Swap Input/Output
- Reset all fields
- Download
	- Source and Target(Tab seperated)
	- Only Target
	- Only Source
- Can view WX chart for Hindi, Telugu and Tamil
- Built completely in Javascript
